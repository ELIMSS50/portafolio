# Luis Felipe Rodriguez Rico — Portfolio

Portfolio personal como Python Developer especializado en optimización de scripts, pipelines de datos y automatización.

## Tecnologías

- HTML5, CSS3, JavaScript (vanilla)
- Fuentes: Fraunces, Sora, JetBrains Mono
- Formulario: Formspree
- Diseño responsive con tema claro/oscuro
- Internacionalización ES/EN

## Estructura

```
├── index.html       # Página principal
├── css/style.css    # Estilos
├── js/main.js       # Lógica (i18n, tema, animaciones)
├── assets/          # Imágenes
├── favicon.svg      # Favicon
└── cv/              # CV en LaTeX (PDF)
```

## Despliegue

Abrir `index.html` en cualquier navegador o servir con cualquier servidor estático:

```bash
npx serve .
```
